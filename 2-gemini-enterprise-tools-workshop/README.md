# Gemini Enterprise Tools Workshop

## What students learn
Students learn how to use core Gemini Enterprise capabilities in practical workflows. They cover image generation for quick starts, Google Search as a grounding tool, structured prompt engineering, file-based context management, deep research, and meta prompting to improve research quality and repeatability.

## Case study
This course uses Cymbal Capital Partners, a fictional investment firm evaluating potential portfolio companies. Learners use Gemini Enterprise tools to investigate markets, assess founders and company background, and produce stronger investment research inputs.
