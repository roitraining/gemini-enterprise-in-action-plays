# Professional Lifestyle Photography

## Overview
This play teaches you how to create high-end, lifestyle marketing assets that seamlessly blend personal identity with corporate branding using Gemini’s Nano Banana 2 model.


### Objectives
In this play, you learn how to use a "Meta-Prompt" to transform a simple human request into a structured, technical prompt.

## Instructions
1. Open Gemini Enterprise in your browser.

2. In the chat, copy and paste the following prompt, then press ENTER.

```
I want to create a high-end athletic lifestyle marketing graphic for my company, Cymbal-Mart, that feels incredibly luxurious. Please write a perfectly engineered image generation prompt for me. The scene needs to feature a person happily working out (based on a headshot called headshot.jpg) and my company logo (CymbalLogo.png), but they must be integrated naturally into the environment—not just placed on top. Focus the description on expensive textures, cinematic lighting, and a sophisticated athletic setting (like an exclusive gym). Make sure the logo appears as a realistic detail on the gear or the architecture.
```

3. Click the Copy response icon at the end of the generated response. 

![Sample screenshot](images/copyresponse.png)

Note: You may also copy the following prompt that we’ve generated for you.

```
Cinematic, photorealistic shot of a person [insert headshot.jpg] happily exercising in an ultra-exclusive luxury gym. The environment features expensive textures like dark oak, polished marble floors, and matte black steel equipment. Illuminated by soft, golden cinematic lighting from large windows. The Cymbal-Mart logo [insert CymbalLogo.png] is naturally embossed on the weight plates and subtly etched into a frosted glass wall behind them. Wearing high-end athletic apparel, 8k resolution, highly detailed.
```

4. Then click New chat to open a new session.

5. In the chat bar, select the Tools icon and then choose Generate images (NB2).

6. Select + Add files and choose Upload files.

7. In the dialog, select headshot.jpg and CymbalLogo.png and click Open.

8. In the chat, paste the prompt you copied, then press ENTER. Save your final result for sharing.
