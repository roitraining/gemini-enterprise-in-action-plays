# Vibe Coding for Instant Deployable Apps

## What students learn
Students learn an end-to-end vibe-coding workflow, from UI sketch to deployed application. They generate a front-end from an image, add CSV ingestion and KPI logic, create interactive charts, introduce a backend that calls Gemini for risk insights, and deploy a working version to Cloud Run.

## Case study
This course uses Cymbal Logistics, a fictional global shipping company modernizing operations. Learners build a Fleet and Shipment Dashboard that transforms daily shipment data into operational KPIs and executive summaries for faster decision-making.
